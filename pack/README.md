# BeautifyJson

Beautify your json string or file use `.NET Framework 4.7.2` and `.NET 9`.

## Formatter Methods

1. Sort primary JSON key order by ASCII.
2. Indent (1 TAB = 4 Blank).

## Usage

```bash
beautify-json test.json
```
